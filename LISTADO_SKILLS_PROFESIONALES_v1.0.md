## 📌 NOTAS DEL EQUIPO

### Metodología de Desarrollo Asistida por IA (≠ Scrum)
- Scrum es metodología **de gestión de proyectos** (sprints, dailies, retrospectivas)
- La **metodología FIC (AI-Assisted Development)** es metodología **técnica** (uso de skills, agents, documentación automática)
- **Ambas son compatibles**: Puedes usar Scrum para gestión + FIC para desarrollo técnico
